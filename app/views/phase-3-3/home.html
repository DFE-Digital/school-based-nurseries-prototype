Apply for a school-based nursery capital grant

Application round open

The deadline to apply is 5:00pm on 30 September 2026.

This service is for:

  • local authorities to submit phase 3 proposals
  • schools to view previous submitted phase 1 or 2 nursery applications

Local authorities can apply for phase 3

Local authorities can use this service to submit a funding proposal setting out project plans for new or expanded school-based nurseries in their area.

Read the guidance on the School-based nursery capital grant 2027 to 2030 (link opens in new tab) before applying.

Before you start

You need a DfE Sign-in account to use this service.

Prepare a separate project plan for every school-based nursery you want to create or expand.

Local authorities can put forward any number of school-based nursery projects within this area. Use the template funding proposal form to gather the necessary information for each project ahead of applying for funding

If you are a school

Individual schools cannot apply for the phase 3 grant. Schools must contact their local authority if they are interested in creating or expanding a nursery. Read the guidance on establishing school-based nursery provision (link opens in new tab).

Phase 1 and 2 applications

Applications for phase 1 and 2 have now closed, and successful applications have been announced here (link opens in new tab). Sign into the service to view and download previous submitted applications.

You need a DfE Sign-in account to use this service.
