Apply for a School-based nursery capital grant

Application round open

The deadline to apply is 5pm on Monday 14 September 2026 .

Use this page to:

  • start a new project plan
  • manage project plans in progress
  • view submitted plans
  • answer childcare sufficiency questions

To be considered for the school-based nursery capital grant, you must include an assessment of local childcare sufficiency needs. Only project plans that are submitted before the deadline will be considered.
