Check eligibility of proposed project

Submit a separate application for each school-based nursery you want to create or expand. Make sure the school's proposed project meets all 5 criteria:

  1. The school's proposed project must increase the total number of childcare places at the school. This includes any places already available on site or nearby, including those run by other providers.
  2. The nursery will be operational and offering new childcare places by September 2030 at the latest. Full capacity does not need to be delivered by 2030.
  3. There is written permission, such as an email or a letter, from the freehold landowner to use the land and carry out the proposed works. Examples of freehold landowners include the local authority, a foundation or trust, or a religious body.
  4. The proposed project must be one of these types:
    • an school-based nursery (SBN) at a state-funded primary school in England that already offers early education, such as reception classes
    • part of a maintained nursery school (MNS) site in England
    • a Best Start Family Hub (BSFH) that already exists or is in the process of being set up - it can be on a school site or elsewhere
  5. If the school already received funding through this grant in either 2024-2025 or 2025-2026, the proposed project must add new places beyond what was delivered in earlier phases, and offer good value for money.
